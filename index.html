Graphite Project

Graphite Project is a simple proof-of-concept game by oxydd, inspired by Mindustry and... that's it.
It started in October 2020 as an even simpler and poorly put together Scratch project which lasted until late 2021 with the last version being 0.9.3.
The modern Graphite Project started in December 2023, originally named "some space game", which was a test of an open randomly generated world within the boundaries of Scratch. Later, in January 2024, this project has been repurposed as a new iteration of the old Graphite Project. Originally named as the beta 1.0 version it was in development until July, when it was changed to alpha 0.10 as passing the requirement for entering the beta stage was seemingly too far in the future.
Right now it is being developed as my passion project, to which i come back once in a while and drop a couple updates.
